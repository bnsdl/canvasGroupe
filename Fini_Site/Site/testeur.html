En complément du tutoriel, nous avons choisi de vous proposer un module de test vous permettant d'appliquer les fonctions précédemment expliquées.
Ces dernières, créées par nos soins, ne sont pas génériques mais vous permettront une 1ère prise en main rapide et efficace pour découvrir l'outil canvas.

Pour des raisons d'ergonomie, nous avons retiré le canvas pour cette taille d'écran, merci de passer sur tablette ou pc afin d'utiliser le testeur.

IN CODE WE TRUST

Les fonctions:

trait(débutX, débutY, finX, finY, couleur)

rectVide(débutX, débutY, tailleX, tailleY, couleur)

rectPlein(débutX, débutY, tailleX, tailleY, couleur)

cercleVide(centreX, centreY, rayon, couleur)

cerclePlein(centreX, centreY, rayon, couleur)

efface(débutX, débutY, tailleX, tailleY)

dessineImage(débutX, débutY, lien)

texte( débutX, débutY, phrase)

degrade(couleur1, couleur2, couleur3)

deplaceOrigine(décalageX, décalageY)

rotationOrigine(angle de rotation de l'origine)
